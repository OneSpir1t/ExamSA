# ExamSA
## GeneralInfo
Описание предметной области для базы данных
#Таблица UserRole
Поскольку пользователи имеют роль таблица UserRole будет иметь следующие поля:
-"ID";
-"Name".
#Таблица Users
Таблица Users будет иметь следующие поля:
-"ID";
-"Name";
-"Login";
-"Password";
-"UserRoleID".
